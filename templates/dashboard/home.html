Meta App Configuration

Connect this workspace with your own Meta app credentials.

Set your app credentials once from UI. Values are saved in project .env so reconnect flow works without code edits.

Why This Matters Multi-user friendly setup

Every operator can set app credentials from dashboard, so onboarding does not depend on editing source code.

Applied Scope OAuth + token operations

New values are used for Meta connect flow and token validation checks after save.

How To Get Values Find META_APP_ID, META_APP_SECRET, META_REDIRECT_URI
  1. Open developers.facebook.com/apps and create/select your app.
  2. Choose Business use-case and add Facebook Login if not already added.
  3. In App Settings -> Basic, copy App ID and paste it as META_APP_ID.
  4. Click Show for App Secret and paste it as META_APP_SECRET.
  5. In Facebook Login settings, set Valid OAuth Redirect URI exactly as your callback URL.
  6. Paste same callback URL here as META_REDIRECT_URI (example: https://your-domain/auth/meta/callback).

After save, use Connect Facebook + Instagram on Accounts page to verify credentials and sync pages.
