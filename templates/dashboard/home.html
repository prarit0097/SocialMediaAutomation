Setup Guide

Get Meta credentials and connect profiles without confusion.

Read this like a checklist. Do one step, then move to next step. If each checkpoint matches, setup is correct.

Goal: Fill 3 fields correctly - META_APP_ID, META_APP_SECRET, META_REDIRECT_URI
Need: Facebook account with access to your Meta app and pages
Result: Accounts page starts showing your FB + IG profiles
Part 1: Create Meta App (what to click)
  1. Chrome me ye website kholo: developers.facebook.com/apps.
  2. Apne Facebook account se login karo (same account jisme aap pages/profiles manage karte ho).
  3. Top-right me Create App button dabao.
  4. Meta poochega app kis kaam ke liye hai: Business choose karo.
  5. App ka naam do (example: My Social Automation App) aur create complete karo.
  6. Ab app dashboard open hoga. Left side me products me Facebook Login add karo.
  7. Facebook Login -> Settings kholo aur Valid OAuth Redirect URIs box dhoondo.
  8. Wahan apna callback URL daalo (example: https://your-domain/auth/meta/callback) aur save karo.
  9. Phir App Settings -> Basic kholo. Yahi screen se App ID aur App Secret milta hai.

Checkpoint: Aapko ek screen dikhni chahiye jahan App ID visible ho aur App Secret ko Show kar sakte ho.

Part 2: Copy -> Paste map (exact)

Rule: Jo value Meta me hai, wahi value yahan paste karo. Extra space ya typo mat karo.

Copy from Meta: App ID -> Paste in dashboard: META_APP_ID
Copy from Meta: App Secret -> Paste in dashboard: META_APP_SECRET
Use your public URL + callback path -> Paste in dashboard: META_REDIRECT_URI
  1. Meta app basic screen se App ID copy karo -> is Home form me META_APP_ID me paste karo.
  2. Wahi screen par Show click karke App Secret copy karo -> META_APP_SECRET me paste karo.
  3. Apna public domain + callback path banao -> META_REDIRECT_URI me paste karo.
  4. Important: Redirect URI ka same exact value Meta ke Facebook Login settings me bhi hona chahiye.

Redirect URI example: https://your-domain/auth/meta/callback (https required, http se Meta login fail ho sakta hai).

Part 3: Save in this dashboard
  1. Home page ke form me tino fields check karo: META_APP_ID, META_APP_SECRET, META_REDIRECT_URI.
  2. Save Meta Configuration button click karo.
  3. 2-3 second wait karo aur green/success message ka wait karo.
  4. Save ke baad Accounts page me jaake Connect Facebook + Instagram click karo.
  5. Meta popup me jitne pages/profiles chahiye, sab select karke continue karo.
  6. Back aane ke baad Refresh List dabao taaki connected rows update ho jaye.

Checkpoint: Save error-free ho, aur Accounts page me rows dikhne lage. Agar error aaye to pehle redirect URI aur App Secret dobara verify karo.

Part 4: Add permissions/scopes (important)
  1. Meta app dashboard kholo, left menu me App Review ya Permissions and Features section me jao.
  2. Search box me permission ka naam type karo (example: pages_show_list), phir Add ya Request button click karo.
  3. Agar Meta "Advanced Access" mange, to us permission ke card me jaake access request complete karo.
  4. Same process neeche diye gaye har scope ke liye repeat karo. Koi scope miss hua to related feature kaam nahi karega.
  5. Sab add karne ke baad setup save karo, phir app me Connect Facebook + Instagram dobara run karo.

Required permissions/scopes for this app

  • pages_show_list
  • pages_read_engagement
  • pages_manage_posts
  • pages_read_user_content
  • read_insights
  • instagram_basic
  • instagram_content_publish
  • instagram_manage_insights
  • business_management (optional but useful in some setups)

Note: some tutorials say instagram_content_publishing; this project uses instagram_content_publish.

What each scope does (simple meaning)

  • pages_show_list -> app ko aapke Facebook Pages ki list dikhata hai (page pick karne ke liye must).
  • pages_read_engagement + read_insights -> page reach/likes/comments analytics read karne ke liye.
  • pages_manage_posts + pages_read_user_content -> Facebook pe post publish karne aur post data read karne ke liye.
  • instagram_basic -> connected Instagram profile ka basic data read karne ke liye.
  • instagram_content_publish -> Instagram post/reel publish karne ke liye (missing hua to scheduling fail ho sakti hai).
  • instagram_manage_insights -> Instagram insights (views, reach, interactions) read karne ke liye.

Quick self-check (before connect)

  • Kya list ke sab scopes added/enabled hain? Agar nahi, pehle complete karo.
  • Kya aapke FB profile ko page par admin/full-control access hai?
  • Kya Instagram profile Professional hai aur FB page se linked hai?
  • Agar sab yes hai, tabhi Connect flow run karo.

Final check before connect: Required scopes + page access + IG professional link tino correct hone chahiye, warna profiles missing, token errors, ya scheduling failure aa sakti hai.

Part 5: Connect and verify profiles in app
  1. Go to Accounts page and click Connect Facebook + Instagram.
  2. Permission popup me all required Facebook Pages + Instagram profiles select karo.
  3. Back in app, click Refresh List.
  4. Check: profiles `Connected Accounts` table me दिखनी चाहिए.
  5. If some profiles missing: reconnect karo and Business Integration selection me un profiles ko tick karo.

Profile rule: Instagram account Professional hona chahiye aur Facebook Page se linked hona chahiye (admin/editor access).

Quick troubleshooting (if stuck)
  • Save failed? Redirect URI format check karo.
  • Profiles missing? Reconnect and select all assets in popup.
  • Catalog only rows? Fresh reconnect + Refresh List run karo.
  • Instagram not posting? Public HTTPS media URL and required scopes check karo.
