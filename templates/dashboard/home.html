Setup Guide

Get Meta credentials and connect profiles without confusion.

Follow these steps in order, then connect and verify all selected pages and Instagram profiles from the Accounts page.

Create Meta App + Login setup
  1. Open developers.facebook.com/apps and click Create App.
  2. Select Business as use case.
  3. Add Facebook Login product.
  4. In Facebook Login settings, add your exact callback URL in Valid OAuth Redirect URIs.
  5. Open App Settings -> Basic, copy App ID and App Secret.
Fill dashboard fields correctly
  1. Paste App ID into META_APP_ID.
  2. Paste App Secret into META_APP_SECRET.
  3. Set META_REDIRECT_URI to value like https://your-domain/auth/meta/callback.
  4. Click Save Meta Configuration.
Scopes and profile linking

Required permissions/scopes

  • pages_show_list
  • pages_read_engagement
  • pages_manage_posts
  • pages_read_user_content
  • read_insights
  • instagram_basic
  • instagram_content_publish
  • instagram_manage_insights
  • business_management (optional but useful in some setups)

Note: some tutorials say instagram_content_publishing; this project uses instagram_content_publish.

Connect + verify in app
  1. Go to Accounts page and click Connect Facebook + Instagram.
  2. During Meta permission popup, select all required Facebook Pages and Instagram profiles.
  3. Click Refresh List and check connected rows + catalog review.
  4. If some rows are missing, reconnect and re-select all assets in Business Integration selection.

Instagram account should be Professional and linked to a Facebook Page with admin/editor access.
