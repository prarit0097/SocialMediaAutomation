Setup Guide

Get Meta credentials and connect profiles step by step.

Read in order. If each checkpoint matches, your setup is production-ready for Postzyo workflows.

Goal: Fill 3 fields correctly - META_APP_ID, META_APP_SECRET, META_REDIRECT_URI
Need: Facebook account with admin/full control on target pages
Result: Connected Accounts + Scheduler + Insights flows become usable
Part 1: Create Meta App (exact clicks)
  1. Open: developers.facebook.com/apps.
  2. Login with same Facebook account that manages your pages/profiles.
  3. Click Create App, choose use case Business.
  4. Name your app (example: Postzyo Business App) and create it.
  5. Add product Facebook Login.
  6. Open Facebook Login -> Settings and add callback in Valid OAuth Redirect URIs.
  7. Open App Settings -> Basic to copy App ID and App Secret.

Checkpoint: You can clearly see App ID and can reveal App Secret in Meta dashboard.

Part 2: Copy -> Paste map (do not change values)
Copy from Meta: App ID -> Paste in dashboard: META_APP_ID
Copy from Meta: App Secret -> Paste in dashboard: META_APP_SECRET
Use public domain + callback path -> Paste in dashboard: META_REDIRECT_URI

Callback example: https://your-domain/auth/meta/callback (must be HTTPS for live use).

Part 3: Save and verify in app
  1. Fill all 3 fields on this page and click Save Meta Configuration.
  2. Go to Accounts and click Connect Facebook + Instagram.
  3. In Meta popup, select all required pages and IG profiles, then continue.
  4. Back in app, click Refresh List.

Checkpoint: Your selected assets appear in Connected Accounts table with correct IDs.

Part 4: Required permissions/scopes

Add these scopes in Meta app

  • pages_show_list
  • pages_read_engagement
  • pages_manage_posts
  • pages_read_user_content
  • read_insights
  • instagram_basic
  • instagram_content_publish
  • instagram_manage_insights
  • business_management (optional but useful)

Some tutorials mention instagram_content_publishing; this project uses instagram_content_publish.

Checkpoint: Missing scope can break connect, scheduling, or insights fetch for specific profiles.

Part 5: Troubleshooting before production
  • Rows missing in Connected Accounts: reconnect and reselect full asset list.
  • Catalog-only status: run fresh connect + Refresh List with current user token.
  • Publish failures: verify profile link (FB page + professional IG) and token health.
  • Insights mismatch: run Force Refresh and check combined FB+IG view.
